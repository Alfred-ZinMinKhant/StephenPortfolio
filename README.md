## ğŸš€ **Getting Started**

### Prerequisites
- Python 3.7+ (for PDF extraction)
- Modern web browser
- Git (for version control)

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/yourusername/min-thadan-portfolio.git
   cd min-thadan-portfolio
   ```

2. **Set up Python environment** (optional, for PDF extraction)
   ```bash
   python -m venv .venv
   source .venv/bin/activate  # On Windows: .venv\Scripts\activate
   pip install -r requirements.txt
   ```

3. **Run locally**
   ```bash
   # Simple HTTP server
   python -m http.server 8080
   
   # Or open index.html directly in your browser
   open index.html
   ```
